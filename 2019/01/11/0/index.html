OC 数据类型

发表于 2019-01-11

1.基本数据类型

数据类型

  • 整型 int
  • 字符型 char
  • 浮点型
    (单精度 float 和 双精度 double)
  • 类型限定修饰词
    (short短型，修饰int、double; long长型，修饰int、double;signed有符号型，修饰int、char;unsigned 无符号型，修饰int、char;)

取值范围

  • int (-2147483648～2147483647)
  • unsigned int (0～4294967295)
  • short：(-32768～32767)
  • unsigned short：(0～65535)
  • long(-2147483648～2147483647)
  • unsigned long(0～4294967295)

存储空间

2.构造类型

  • 数组类型 常见NSArray、NSMutableArray

  • 结构体类型 例如

  • 共用体类型 (一个能在同一个存储空间存储不同数据的类型)

3.指针类型

  • 所有的系统类, 自定义类都是指针

4.数据对象类型

  • nil (表示一个Objctive-C对象，这个对象的指针指向空)
  • Nil (定义一个指向空的类是Class，而不是对象）
  • NULL (指向其他类型的空指针，如一个c类型的内存指针)
  • NSNull (在集合对象中，表示空值的对象)
  • 数据包装类型
